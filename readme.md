**Velo**是一个 **Vue3** 组件库开发环境框架,采用最新的 `Vite4+TypeScript` 为技术栈,支持`按需加载`,`单元测试`,`自动打包与发布`等功能,让我们能更专注于业务组件的开发。

## 官网地址

[Velo](https://ajh-velo.github.io/velo/)

## 快速启动

- 安装 pnpm

```
npm install pnpm -g
```

- 安装依赖

```
pnpm install
```

- 打包组件库与工具库

```
pnpm run build:easyest
```

- 启动测试项目

```
pnpm run easyest:dev
```

- 启动静态文档站点

```
pnpm run docs:dev
```
